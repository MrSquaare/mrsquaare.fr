## Getting started

### Prerequisites

1. [Install Node.js](https://nodejs.org/en/download/)

### Installation

1. Clone the repository:

```shell script
git clone https://github.com/MrSquaare/mrsquaare.fr.git
```

2. Install dependencies:

```shell script
npm install
```

## Developing

Start the development server:

```shell script
npm run dev
```

## Testing

Lint the code:

```shell script
npm run lint
```

Format the code:

```shell script
npm run format
```
