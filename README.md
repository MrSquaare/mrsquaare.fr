## Getting started

### Prerequisites

1. [Install Node.js](https://nodejs.org/en/download/)
2. [Install pnpm](https://pnpm.io/installation)

### Installation

1. Clone the repository:

```shell script
git clone https://github.com/MrSquaare/mrsquaare.fr.git
cd mrsquaare.fr
```

2. Install dependencies:

```shell script
pnpm install
```

3. Build the project:

```shell script
pnpm build
```

## Using

Start the servers:

```shell script
pnpm start
```
